About PyPass

Our mission

Our mission is to add the security and versatility of a paid password manager, without the huge hole in your wallet and without the barriers between operating systems.


Supported Operating Systems

Offically supported Operating Systems:

  • Android
  • Windows 11 (untested)
  • Windows 10 (lightly tested)
  • RHEL-based linux distributions
  • Debian-based linux distributions
  • Arch-based linux distributions (untested)
  • macOS (lightly tested on older versions)

NOTE: You may be warned against using system packages for PyPass. This is because they are self-signed, and will be fixed when we can get the funds


Things we want to add

  • Add support for iOS
  • Sign packages with signing key on all platforms

All funds donated to PyPass will go toward the advancement of PyPass
